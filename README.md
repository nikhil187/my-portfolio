## Project Description
This project is a self-profile application built using ReactJS and Bootstrap. It showcases my personal information, including my education, skills, and experiences. The application is designed to be responsive and user-friendly, utilizing Bootstrap for layout and styling.

### Features:
- **Profile Page**: Displays personal information, including a profile picture, bio, and social media links.
- **Classmates Page**: Lists classmates with their favorite foods and colors, dynamically generated from an array of objects.
- **Like Button**: Each classmate card includes a like button that increments the like count when clicked.

## People Array
The application uses an array named `People` to store information about classmates. Each object in the array contains the following properties:
- `name`: The person's name.
- `favoriteFood`: The person's favorite food.
- `favoriteColor`: The person's favorite color.

This array is used to dynamically generate classmate cards using the `.map()` function, ensuring that the UI updates automatically when the data changes.
